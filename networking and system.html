Networking and Systems

COURSE DESCRIPTION

Somewhere there might be a field of interest in which the order of presentation of topics is well agreed upon. Computer networking is not it. There are many interconnections in the field of networking, as in most technical fields, and it is difficult to find an order of presentation that does not involve endless “forward references” to future chapters; this is true even if – as is done here – a largely bottom-up ordering is followed. I have therefore taken a different approach. Local Area Networks, or LANs, are the “physical” networks that provide the connection between machines within, say, a home, school or corporation. LANs are, as the name says, “local”; it is the IP, or Internet Protocol, layer that provides an abstraction for connecting multiple LANs into, well, the Internet. Finally, TCP deals with transport and connections and actually sending user data.


PREREQUISITES

Computer networking is a vast field of study, although getting into the basics can be fairly simple if you are willing to make the effort. Learning basic networking is fairly straightforward but gaining a depth of knowledge and proper hands-on experience takes a lot of time and dedication. Having keen analytical skills is an important part to learning networking. Setting up, installing, and maintaining computer networks only scratches the surface of what you'll be required to do; you also need to have the proper skills necessary to quickly troubleshoot and solve complex problems and figure out more efficient ways to perform your work.


COURSE CONTENTS

● An Overview of Networks

In simplest terms, a computer network consists of two or more connected computers. This connection is twofold: (a) physical, through wires, cables, and wireless media (the atmosphere with, say, cell phones), and (b) logical, through the transport of data across the physical media. We discuss the components required to make the physical connection in several parts of this book; notably, Hours 4, 10, and 23. The logical connections are discussed throughout the book.


● Ethernet Basics

A standard Ethernet network can transmit data at a rate up to 10 Megabits per second (10 Mbps). Other LAN types include Token Ring, Fast Ethernet, Gigabit Ethernet, 10 Gigabit Ethernet, Fiber Distributed Data Interface (FDDI), Asynchronous Transfer Mode (ATM) and LocalTalk.

● Advanced Ethernet

Ethernet is a family of wired computer networking technologies commonly used in local area networks (LAN), metropolitan area networks (MAN) and wide area networks (WAN). It was commercially introduced in 1980 and first standardized in 1983 as IEEE 802.3. Ethernet has since been refined to support higher bit rates, a greater number of nodes, and longer link distances, but retains much backward compatibility. Over time, Ethernet has largely replaced competing wired LAN technologies such as Token Ring, FDDI and ARCNET.


● Wireless LANs

A wireless LAN (WLAN) is a wireless computer network that links two or more devices using wireless communication to form a local area network (LAN) within a limited area such as a home, school, computer laboratory, campus, or office building. This gives users the ability to move around within the area and remain connected to the network. Through a gateway, a WLAN can also provide a connection to the wider Internet.


● Other LANs

In general, there are two types of LANs: client/server LANs and peer-to-peer LANs. A client/server LAN consists of several devices (the clients) connected to a central server. The server manages file storage, application access, device access, and network traffic.


● Links

A link (short for hyperlink) is an HTML object that allows you to jump to a new location when you click or tap it. Links are found on almost every webpage and provide a simple means of navigating between pages on the web. Links can be attached to text, images, or other HTML elements.


● Packets

Data sent over a network is divided into smaller segments called packets. Learn how Internet packets work, what an IP packet is, and what datagram means.


● Abstract Sliding Windows

Retransmit-on-timeout generally requires sequence numbering for the packets, though if a network path is guaranteed not to reorder packets then it is safe to allow the sequence numbers to wrap around surprisingly quickly. However, as the no-reordering hypothesis does not apply to the Internet at large, we will assume conventional numbering. Data will be the Nth data packet, acknowledged by ACK.


● IP version 4

Internet Protocol version 4 (IPv4) is the fourth version of the Internet Protocol (IP). It is one of the core protocols of standards-based internetworking methods in the Internet and other packet-switched networks. IPv4 was the first version deployed for production on SATNET in 1982 and on the ARPANET in January 1983. It is still used to route most Internet traffic today,[1] despite the ongoing deployment of a successor protocol, IPv6.


● IPv4 Companion Protocols

IP is the keystone of the Internet, but it occupies that position with a little help from its friends. DNS translates human-readable host names, such as intronetworks.cs.luc.edu to IP addresses. ARP translates IPv4 addresses to Ethernet addresses, for destinations on the same LAN.


● IPv6

Internet Protocol version 6 (IPv6) is the most recent version of the Internet Protocol (IP), the communications protocol that provides an identification and location system for computers on networks and routes traffic across the Internet. IPv6 was developed by the Internet Engineering Task Force (IETF) to deal with the long-anticipated problem of IPv4 address exhaustion. IPv6 is intended to replace IPv4. In December 1998, IPv6 became a Draft Standard for the IETF, who subsequently ratified it as an Internet Standard on 14 July 2017.


● IPv6 Additional Features

In contrast to IPv4, IPv6 uses 4 times more bits to address a device on the Internet. This much of extra bits can provide approximately 3.4×1038 different combinations of addresses. This address can accumulate the aggressive requirement of address allotment for almost everything in this world. According to an estimate, 1564 addresses can be allocated to every square meter of this earth.


Frequently Asked Questions

What is a Network?

Network is defined as a set of devices connected to each other using a physical transmission medium. A network is defined as a group of devices connected to each other through physical transmission media.

What is a Node?

Two or more computers are directly connected by optical fiber or other cables. A node is a point at which a connection is established. It is a network component used to send, receive, and forward electronic information. The devices connected to the La network are also called nodes. Assuming that there are 2 computers, 2 printers, and a server connected to the network, we can say that there are five nodes on the network.

What are Routers?

A router is a network device that connects two or more network segments. It is used to transfer information from source to target. Routers send information in the form of data packets, when these data packets are forwarded from one router to another. , And then the router reads the network address in the data packet and determines the target network.

What is HTTP and what port does it use?

HTTP is a hypertext transfer protocol responsible for web content. Many websites use HTTP to deliver web content and enable hypertext display and navigation. This is the main protocol, and TCP port 80 is used here.

What is DNS?

Domain Name Server (DNS) is a non-professional language, we can call it the phone book on the Internet. All public IP addresses and their host names are stored in DNS and then resolved to the corresponding IP address. The domain name is easy to remember and recognize, but the computer is a machine that does not understand human language. It only understands the language of the IP address used for data transmission.

Define Static IP and Dynamic IP?

When a device or computer is assigned a specific IP address, it is assigned a static IP name and assigned as a permanent address by the ISP.

Define IEEE in the networking world?

IEEE stands for Institute of Electrical and Electronic Engineering. It is used to design or develop network standards.

Explain the difference between baseband and broadband transmission?

Baseband Transmission: A single signal consumes the whole bandwidth of the cable. Broadband Transmission: Multiple signals of multiple frequencies are sent simultaneously.

What is a Decoder?

Decoder is a circuit that converts the encoded data to its actual format. It converts the digital signal into an analog signal.
